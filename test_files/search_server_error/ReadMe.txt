
To test programm interactive
--> from folder with program "search_server_error.py" 
--> go to folder: "cd test_files/search_server_error/"
--> Start the programm in the console: "python3 ../../search_server_error.py"
--> Enter into console 
   --> The path for the test input file (month to check .log Data):     "test_input.txt"
   --> The path for the test output file:                                            "test_output.txt"

--> from folder with program "search_server_error.py" 
--> go to folder: "cd test_files/search_server_error/"
--> Start the programm in the console: "python3 ../../search_server_error.py  test_input.txt test_output.txt"
